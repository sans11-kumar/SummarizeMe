How to use the extension

  1. Navigate to any webpage you want to summarize
  2. Click the extension icon
  3. Press "Summarize This Page"
  4. Wait for the summary to be generated
  5. Ask follow-up questions in the chat interface

Local LLM Setup

  1. Download and install LM Studio
  2. Load the deepseek-r1-distill-qwen-7b model
  3. Start the local server in LM Studio
  4. Use the extension to summarize content

API Keys Security

Your API keys are stored locally in your browser's secure storage. When encryption is enabled, keys are encrypted before storage for additional security.
